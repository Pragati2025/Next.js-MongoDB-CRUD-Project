# Next.js + MongoDB CRUD Project

A full-stack CRUD application built with **Next.js 16 (App Router)**, **MongoDB**, and **Mongoose**.  
It allows users to **Create, Read, Update, and Delete** persons through a simple, responsive UI with a professional white, blue, and grey theme.

## Features

- Add, view, edit, and delete persons
- Responsive design
- REST API built with Next.js API routes
- MongoDB integration using Mongoose

## Tech Stack

- Frontend: Next.js 16, React.js, TypeScript, CSS
- Backend: Next.js API Routes, Node.js
- Database: MongoDB

## Getting Started

1. Clone the repository  
   `git clone https://github.com/Pragati2025/Next.js-MongoDB-CRUD-Project.git`
2. Install dependencies  
   `npm install`
3. Set `.env.local` with your MongoDB URI:  
   `MONGO_URI=mongodb://localhost:27017/personcrud`
4. Run the app  
   `npm run dev`  
   Open [http://localhost:3000](http://localhost:3000)
